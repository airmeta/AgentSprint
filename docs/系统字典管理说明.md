# 系统字典管理说明

字典管理用于维护系统内可复用的枚举分类和值，当前作为“系统管理”下的独立功能提供。

## 管理入口

- 管理端页面：`/system/dictionaries`
- 后端菜单：`SystemDictionaries`
- 菜单图标：`lucide:book-open-text`
- 菜单权限码：`System:Dictionary:Manage`

## 数据模型

- 字典类型表：`sys_dictionary_type`
- 字典项表：`sys_dictionary_item`

字典类型字段包含 `Code`、`Name`、`Description`、`Sort`、`Status`。字典项字段包含 `DictionaryTypeId`、`Code`、`Name`、`Description`、`Sort`、`Status`。

## 接口

接口统一挂载在 `/system`，需要登录后访问。

- `GET /system/dictionary-types`
- `POST /system/dictionary-types`
- `DELETE /system/dictionary-types/{id}`
- `GET /system/dictionary-items?dictionaryTypeId={id}`
- `POST /system/dictionary-items`
- `DELETE /system/dictionary-items/{id}`

## 维护规则

- 字典类型 `Code` 全局唯一，列表按 `Sort`、`Code` 稳定排序。
- 字典项必须归属到有效字典类型，同一字典类型内 `Code` 唯一。
- 删除字典类型时会同步软删除该类型下的字典项。
- 删除单个字典项不影响字典类型和同类型其他条目。
- `Status` 保留为启停状态，供后续业务下拉按状态过滤。
