# Skill配置管理说明

## 功能范围

Skill 配置用于维护可复用的 Codex/Agent 指令内容。启用状态的 Skill 可以在创建或编辑端配置时选择，也可以在新建或编辑需求时选择。

## 使用方式

1. 在项目管理菜单进入 `Skill配置`。
2. 新增 Skill，填写唯一编码、名称、说明和完整指令内容。
3. 在多端管理中新建或编辑端时，选择该端需要绑定的 Skill。
4. 新建需求选择端后默认继承该端已选择的 Skill，也可以在需求表单中调整。

## 规则

- Skill 编码全局唯一。
- 停用的 Skill 不再出现在端和需求选择项中。
- 后端会校验端和需求提交的 Skill 必须存在且处于启用状态。
- 编辑端或需求时，如果请求未传 `skillIds`，后端会保留原有选择；如果显式传空数组，则清空选择。
