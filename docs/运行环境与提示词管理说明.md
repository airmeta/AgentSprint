# 运行环境与提示词管理说明

## 功能范围

新增“全局配置”一级菜单，包含“环境管理”和“提示词设置”两个入口。

- 环境管理用于维护测试、预发、生产等环境的访问地址、部署路径、发布包路径和容器端口映射。
- 环境可以关联项目、端和模块，后续适配多端应用、微服务模块和独立部署单元时可沿用同一结构。
- 提示词设置按 Agent 环境拆分，目前仅开放 Codex 模板维护。
- Claude Code 和其他环境在前端展示为“正在开发”，后端会拒绝保存非 Codex 模板。

## 运行环境字段拆解

运行环境主数据：

| 字段 | 说明 | 示例 |
| --- | --- | --- |
| ProjectId | 所属项目，可为空表示全局环境 | AgentSprint |
| EndpointId | 所属端，可为空 | admin / api / mcp |
| ModuleId | 所属模块，可为空 | system / sprint |
| Code | 环境编码，同一项目范围内唯一 | test |
| Name | 环境名称 | 测试环境 |
| EnvironmentType | 环境类型 | test |
| FrontendUrl | 前端访问地址 | `http://192.168.80.101:5999` |
| ApiBaseUrl | API 地址 | `http://192.168.80.101:5000` |
| FrontendProxyApiUrl | 前端代理 API | `http://192.168.80.101:5999/api` |
| McpEndpoint | MCP 地址 | `http://192.168.80.101:5010/mcp` |
| DeployRoot | 部署根目录 | `/opt/agentsprint-deploy` |
| DockerDirectory | Docker Compose 目录 | `/opt/agentsprint-deploy/docker` |
| RemotePackagePath | 远端发布包 | `/opt/agentsprint-deploy/agentsprint-docker-deploy.tgz` |
| ComposeFilePath | Compose 文件 | `/opt/agentsprint-deploy/docker/docker-compose.yml` |
| LocalPackagePaths | 本地发布包路径，按行维护多条 | `F:\AI\AgentSprint\agentsprint-docker-deploy.tar` |

容器映射明细：

| 容器 | 宿主端口 | 容器端口 |
| --- | --- | --- |
| agentsprint-admin | 5999 | 80 |
| agentsprint-api | 5000 | 5000 |
| agentsprint-mcp | 5010 | 5010 |

## API

- `GET /system/runtime-environments?projectId=&endpointId=&moduleId=` 查询运行环境。
- `POST /system/runtime-environments` 新增或更新运行环境。
- `DELETE /system/runtime-environments/{id}` 删除运行环境，并同步删除其容器映射。
- `GET /system/runtime-environment-containers?runtimeEnvironmentId=` 查询容器映射。
- `POST /system/runtime-environment-containers` 新增或更新容器映射。
- `DELETE /system/runtime-environment-containers/{id}` 删除容器映射。
- `GET /system/prompt-templates?agentEnvironment=codex` 查询 Codex 提示词模板。
- `POST /system/prompt-templates` 新增或更新 Codex 提示词模板。
- `DELETE /system/prompt-templates/{id}` 删除提示词模板。

## 初始化数据

数据库初始化会创建以下默认数据：

- 测试环境 `test`，包含用户提供的前端、API、MCP、部署目录、Compose 文件和本地发布包路径。
- 三条容器映射：`agentsprint-admin`、`agentsprint-api`、`agentsprint-mcp`。
- Codex 默认提示词模板 `task_execution`。
- 管理端菜单：`/global-config/environments`、`/global-config/prompt-templates`，分别复用环境管理和提示词设置页面。
- 权限码：`System:RuntimeEnvironment:Manage`、`System:PromptTemplate:Manage`。
